Accordion

Displays collapsible content panels for presenting information in a limited amount of space

The accordion component builds on top of the collapse directive to provide a list of items, with collapsible bodies that are collapsed or expanded by clicking on the item's header.

Usage

// RECOMMENDED (doesn't work with system.js)
import { AccordionModule } from 'ngx-bootstrap/accordion';
// or
import { AccordionModule } from 'ngx-bootstrap';

@NgModule({
  imports: [AccordionModule.forRoot(),...]
})
export class AppModule(){}

API Reference

AccordionPanelComponent

Selector accordion-group, accordion-panel

Inputs

heading
Type: string

Clickable text in accordion's group header, check accordion heading below for using html in header

isDisabled
Type: boolean

if true — disables accordion group

isOpen
Type: boolean

Is accordion group open or closed

panelClass
Type: string

Provides an ability to use Bootstrap's contextual panel classes (panel-primary, panel-success, panel-info, etc...). List of all available classes available here

AccordionConfig

Configuration service, provides default values for the AccordionComponent.

Properties

closeOthers
Type: Boolean
Default value: false

Whether the other panels should be closed when a panel is opened
