Menus

Input Properties:

content= any 对菜单应使用的内容元素的引用。

id= string An id for the menu.

side="left | right" default left"

swipeEnabled=boolean If true, swiping the menu is enabled. Default true.

persistent=boolean If true, the menu will persist on child pages.

type="overlay | reveal | push"

Output Events:

ionClose ionDrag ionOpen

reveal is the default type for iOS mode.

The default type for both Material Design and Windows mode is overlay

The menu type can be changed in the app's config via the menuType property, or passed in the type property on the

imports: [
    IonicModule.forRoot(MyApp,{{'{'}}
        menuType: 'push',
        platforms: {{'{'}}
            ios: {{'{'}}
                menuType: 'overlay',
            {{'}'}}
        {{'}'}}
    {{'}'}})
]
